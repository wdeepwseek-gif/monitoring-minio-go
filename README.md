# monitoring-minio-go
Сервис мониторинга MinIO,который отслеживает события в бакетах (создание/удаление файлов) и отправляет уведомления в Telegram. Подходит для мониторинга хранилища, аудита операций и командной работы. Бот подключается к MinIO через API,фильтрует события по настройкам и автоматически распределяет уведомления по топикам.
